Robotics — Mobility & Control
▸ Magic Induction Car Built

A multi-functional robotic vehicle built using an Arduino UNO, L298N motor driver, four DC motors, five IR sensors, and an HC-05 Bluetooth module. It runs in four different modes: Follow Mode, Line Follow Mode, Dodge Mode, and Bluetooth Remote Control Mode (each detailed below).

Two front IR sensors detect and follow objects, two bottom IR sensors detect lines for path following, and a top IR sensor handles obstacle avoidance. The project demonstrates robotics, sensor integration, motor control, and wireless communication.

Lab Notes — Electronics Experiments
▸ DHT11 Temperature & Humidity Logged

Uses a DHT11 sensor to measure environmental temperature and humidity. The sensor sends data to an Arduino, which processes and displays the readings — useful for weather monitoring and environmental studies.

▸ LCD + Keypad Logged

Demonstrates user interaction through a keypad and visual output on an LCD display. Users enter information via the keypad, and the LCD shows menus, messages, or entered data — an introduction to basic UI design in embedded systems.

▸ PIR Motion Detection Logged

Uses a Passive Infrared sensor to detect human movement. When motion is detected, the system can activate alarms, lights, or notifications — commonly used in security systems.

▸ Ultrasonic Distance Measurement Logged

Measures the distance between the sensor and nearby objects. The sensor emits ultrasonic waves and calculates distance based on the return time of the echo signal — useful for obstacle detection and robotics.

▸ LDR Light Detection Logged

Uses a Light Dependent Resistor to measure ambient light intensity. The system can automatically respond to changes in lighting conditions, such as turning lights on or off.

▸ HC-05 Bluetooth Control Experiments Logged

Explores wireless communication using the HC-05 Bluetooth module — demonstrating how a smartphone can send commands to Arduino systems, enabling remote control of robots, devices, and automation systems.
